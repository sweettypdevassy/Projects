# Node Application Containerized with Docker and Jenkins Integration

This project demonstrates how to containerize a Node.js application using Docker and automate the build and deployment process with Jenkins. Each build is triggered in Jenkins, which creates a Docker image, pushes it to Docker Hub, and notifies the result via email.
